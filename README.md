# AI_Solemne_2
**AI_Solemne_2** is a software project for detecting salmon heads and tails using AI.  
The software has been tested on a laptop with **16 GB RAM** and **NVIDIA RTX 3050 GPU**.
### Requirements
- **Operating System:** Linux  
  *(Code tested on WSL Ubuntu 24.04.1 LTS)*
- **Python:** 3.12 or higher  
- **Pip** installed
### Dependencies
This project works with `requirements.txt` to manage dependencies
1. Clone the repository
```bash
https://github.com/Vinbu/AI_Solemne_2.git
```  
3. Install dependencies
```bash
pip install -r requirements.txt
```  
This will install dependencies listed in requirements.txt  


4. Download Images and Labels 
Click the link to download the dataset used in this project:
https://drive.google.com/drive/folders/1fYlchR5h9JAGNJMpDIGHWBPnX7_6bIHl?usp=sharing  
After downloading, copy the folders content and singular_images into the root of the cloned repository.


5. Run code
In the repository root run:
```bash
jupyter run solemne_2_notebook.ipynb
```
